ShExC basics¶

Cardinality¶

Shape school:Enrollee has a Triple Constraint for triples with property ex:hasGuardian and value of type IRI.

Matches exactly one such triple (the default).

 
school:Enrollee {
    ex:hasGuardian IRI
}

+ - Matches one or more such triples.

 
    "type": "TripleConstraint", "min": 1, "max": -1,

    ex:hasGuardian IRI +

* - Matches zero or more such triples.

 
    "type": "TripleConstraint", "min": 0, "max": -1,

    ex:hasGuardian IRI *

{m} - Matches exactly two such triples.

 
    "type": "TripleConstraint", "min": 2, "max": 2,

    ex:hasGuardian IRI {2,}

{m,n} - Matches at least one and no more than two such triples.

 
    "type": "TripleConstraint", "min": 1, "max": 2,

    ex:hasGuardian IRI {1,2}
